Principais comandos R em estatística

- Para atribuir variáveis:

  • 'x <- 5' ou 'x = 5' ou 'x <<- 5'

    • - Para resolver operações matemáticas:

  • '+, -, *, /, ^' (adição, subtração, multiplicação, divisão, potenciação; respectivamente)

    • - Para mostrar a tabela na tela:

  • 'print(nome_tabela)'

    • - Para calcular a variância:

  • 'variancia <- var(nome_tabela$Item)'

    • - Para calcular o desvio padrão:

  • 'desvio_padrao <- sd(nome_tabela$Item)'

    • - Para resolver funções matemáticas :

  • 'sqrt(), exp(), log(), sin(), cos(), tan()', entre outros.

    • - Para calcular vetores e matrizes:

  • 'c()', 'vector()', 'matrix()', 'array()'

    • - Para listar listas:

  • 'list()'

    • - Para fazer Data Frames:

  • 'data.frame()'

    • - Para acessar estruturas de controle:

  • 'if', 'else', 'for', 'while', 'repeat', 'break', 'next'.

    • - Para realizar funções definidas pelo usuário:

  • 'function()'

    • - Para fazer gráficos básicos:

  • 'plot()', 'hist()', 'boxplot()', 'barplot()', 'pie()'

    • - Para manipulação de arquivos e diretórios:

  • 'read.csv()', 'write.csv()', 'file.choose()', 'setwd()', 'getwd()'

    • - Para manipulação de pacotes:

  • 'install.packages()', 'library()', 'require()'

    • - Para estatísticas descritivas:

  • 'mean()', 'median()', 'sd()', 'var()', 'cor()', 'cov()'

    • - Para realizar testes estatísticos:

  • 't.test()', 'wilcox.test()', 'chisq.test()', 'fisher.test()', 'cor.test()'

    • - Para realizar modelagem estatística: :

  • 'lm()', 'glm()', 'aov()', 'predict()', 'anova()'
